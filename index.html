关于本镜像站

本镜像站提供优化的 jsDelivr 内容分发网络（CDN）加速服务，专为中国大陆用户优化访问体验。

官方域名： https://jsdelivr.linexic.top。 站点托管于 Vercel，并且完全开源，代码可在 GitHub 获取。

快速搜索 npm 包

如何使用本镜像站？

如果您需要访问 jsDelivr 提供的加速资源，例如：

https://cdn.jsdelivr.net/npm/jquery@4.0.0-beta.2/dist/jquery.min.js

请将 cdn.jsdelivr.net 替换为 jsdelivr.linexic.top，即可获得优化后的加速链接：

https://jsdelivr.linexic.top/npm/jquery@4.0.0-beta.2/dist/jquery.min.js

此方法适用于所有 jsDelivr 支持的 CDN 资源，保证访问稳定性和速度。

如何进入项目文件夹？

访问如下格式的链接：

https://jsdelivr.linexic.top/npm/包名@版本/

请将 包名 和 版本号替换为npm包的包名和版本号，即可获得该项目的文件夹链接：

https://jsdelivr.linexic.top/npm/jquery@4.0.0-beta.2/

使用规范

为了确保服务的稳定性与合规性，使用本镜像站时请遵守以下规定：

  • 不得 加速或传播违反中国大陆法律法规的内容，包括但不限于色情、暴力、政治敏感信息等。

  • 如需清除缓存，请参考 jsDelivr 官方指南。
