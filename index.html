泽客CDN反代

域名：cdn.zeinklab.com

本服务提供常用前端资源库的反向代理加速，适用于开发环境和生产环境。所有路径均支持通配符匹配，可直接替换原有CDN链接。

重要限制：直接访问根路径可查看此页面；任何以 .pdf 结尾的请求将被阻止（HTTP 403）。

使用示例

HTML 中替换 CDN 链接

原始链接：

<script src="https://cdn.jsdelivr.net/npm/vue@3.3.4/dist/vue.global.js"></script>

替换为：

<script src="https://cdn.zeinklab.com/npm/vue@3.3.4/dist/vue.global.js"></script>

API 调用示例

访问 Gemini API：

POST https://cdn.zeinklab.com/gemini/v1/models/gemini-pro:generateContent

Gravatar 头像调用

生成用户头像（200像素，默认神秘人头像）：

<img src="https://cdn.zeinklab.com/avatar/EMAIL_MD5_HASH?s=200&d=mp" alt="用户头像">

规则说明

规则类型匹配模式响应优先级

注意：所有代理路径均为 200 状态码重写，浏览器地址栏不会变化。不支持自定义路径添加或修改，如需调整请重新部署配置。

性能建议

  • 生产环境建议配置适当的缓存策略
  • HTTPS 必须启用，所有代理目标均使用 HTTPS 协议
  • 建议开启 HTTP/2 或 HTTP/3 以提升加载速度
  • 对于高并发场景，考虑添加边缘缓存层
