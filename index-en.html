About This Mirror

This mirror provides an optimized jsDelivr Content Delivery Network (CDN) acceleration service, specifically for users in China.

Official domain: https://jsdelivr.linexic.top. The site is hosted on Vercel and is fully open-source. You can find the code on GitHub.

How to Use This Mirror?

If you need to access accelerated resources from jsDelivr, for example:

https://cdn.jsdelivr.net/npm/jquery@4.0.0-beta.2/dist/jquery.min.js

Simply replace cdn.jsdelivr.net with jsdelivr.linexic.top to get an optimized acceleration link:

https://jsdelivr.linexic.top/npm/jquery@4.0.0-beta.2/dist/jquery.min.js

This method applies to all resources supported by jsDelivr, ensuring stable and fast access.

How to Access Project Files?

Use the following link format:

https://jsdelivr.linexic.top/npm/package@version/

Replace package and version with the actual npm package name and version to get the project directory link:

https://jsdelivr.linexic.top/npm/jquery@4.0.0-beta.2/

Usage Rules

To ensure service stability and compliance, please follow these rules:

  • Do not use this mirror to accelerate or distribute content that violates Chinese laws, including but not limited to pornography, violence, or politically sensitive information.
  • The mirror provides stable acceleration but does not guarantee long-term availability.
  • To clear cache, please refer to jsDelivr's official guide.
